# 🔍 DeepFake Detection System

A **Streamlit-based web application** that detects deepfake content from **images, audio, and videos** using deep learning models.

---

## 📌 Overview
This project detects AI-generated or manipulated media by analyzing both visual and audio data. It provides a simple interface where users can upload files and get real-time predictions.

---

## 🚀 Features
- Image Deepfake Detection  
- Audio Deepfake Detection  
- Video Deepfake Detection  
- Real-time prediction with confidence score  
- Final classification as **REAL or FAKE**  

---

## 🛠️ Technologies Used
- Python  
- Streamlit  
- PyTorch  
- HuggingFace Transformers  
- OpenCV  
- Librosa  

---

## ⚙️ System Workflow
1. Upload file (image/audio/video)  
2. Preprocess data  
3. Extract features using models  
4. Analyze and compute score  
5. Display result  

---

## 💻 Installation
git clone https://github.com/your-username/DeepFake-detection-System.git
cd DeepFake-detection-System
pip install -r requirements.txt

## ▶️ Run the Application
streamlit run main.py
